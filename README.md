# MooMooBot
a full functional discord bot to send moomoo.io bots

# Running the bot

```
git clone https://github.com/NuroC/MooMooBot
cd MooMooBot
npm install
npm run index.js
```
# commands
```
!send <server> <name>
ex: !send 8:0:0 Nuro

!token (generates a token to test if there's nothing wrong)
```
