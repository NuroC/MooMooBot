# MooMooBot
a full functional discord bot to send moomoo.io bots

# commands
```
!send <server> <name>
ex: !send 8:0:0 Nuro

!token (generates a token to test if there's nothing wrong)

!fill server <dev/sandbox/normal>
ex: !fill 8:0:0 normal

!remove
removes all fill bots

!crash server
ex: !crash 8:0:0

!info <server>
ex: !info 8:0:0
```
